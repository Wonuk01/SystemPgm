# Linux 기본 명령어 정리

## 1. 도움말 확인 방법

리눅스에서 명령어 사용법을 확인하는 방법

| 명령어 | 설명 |
|---|---|
| `man 명령어` | 매뉴얼 페이지 확인 |
| `info 명령어` | GNU info 문서 확인 |
| `명령어 --help` | 간단한 옵션 설명 확인 |

예시
```
man ls
ls --help
```

---

## 2. 관리자(root) 관련 명령어

| 명령어 | 설명 |
|---|---|
| `passwd root` | root 계정 비밀번호 설정 |
| `apt-get update` | 패키지 목록 업데이트 |
| `apt install gcc` | gcc 컴파일러 설치 |

예시
```
passwd root
apt-get update
apt install gcc
```

---

## 3. 디렉토리 이동

Home 디렉토리로 이동

```
cd ~
```

---

## 4. 경로(Path) 의미

| 기호 | 의미 |
|---|---|
| `.` | 현재 디렉토리 |
| `..` | 부모 디렉토리 |
| `~` | Home 디렉토리 |

# Linux ls 명령어 정리

## 1. ls

현재 디렉토리의 파일 / 폴더 목록 출력

```
ls
```

---

## 2. ls -l

파일 정보를 **상세 목록(Long format)** 으로 출력

```
ls -l
```

출력 정보
- 파일 권한
- 소유자 / 그룹
- 파일 크기
- 수정 시간
- 파일 이름

---

## 3. ls -al

**숨김 파일 + 상세 정보** 함께 출력

```
ls -al
```

옵션 의미

| 옵션 | 설명 |
|---|---|
| `-a` | 숨김 파일 포함 (`.`으로 시작하는 파일) |
| `-l` | 상세 정보 출력 |

---

## 4. ll

`ll`은 기본 명령어가 아니라 **alias로 만든 단축 명령어**

보통 다음과 같이 설정됨

```
alias ll='ls -alF'
```

사용

```
ll
```

---

## 5. alias

자주 사용하는 명령어를 **짧게 등록하는 기능**

| 명령어 | 설명 |
|---|---|
| `alias` | 등록된 alias 확인 |
| `alias 이름='명령어'` | alias 등록 |
| `unalias 이름` | alias 삭제 |

예시

```
alias ll='ls -alF'
alias
unalias ll
```

영구 적용 (예: bash)

```
~/.bashrc
```
