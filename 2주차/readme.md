# Linux 기본 명령어 정리

## 1. 도움말 확인 방법

리눅스에서 명령어 사용법을 확인하는 방법

* `man 명령어` : 매뉴얼 페이지 확인
* `info 명령어` : GNU info 문서 확인
* `명령어 --help` : 간단한 옵션 설명 확인

예시

```
man ls
ls --help
```

---

## 2. 관리자(root) 관련 명령어

root 계정 비밀번호 설정

```
passwd root
```

패키지 목록 업데이트

```
apt-get update
```

gcc 컴파일러 설치

```
apt install gcc
```

---

## 3. 디렉토리 이동

Home 디렉토리로 이동

```
cd ~
```

---

## 4. 경로(Path) 의미

| 기호   | 의미        |
| ---- | --------- |
| `.`  | 현재 디렉토리   |
| `..` | 부모 디렉토리   |
| `~`  | Home 디렉토리 |
| `/`  | Root 디렉토리 |
